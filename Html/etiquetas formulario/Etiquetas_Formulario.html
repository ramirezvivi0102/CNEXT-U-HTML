checkbox

El checkbox es una caja le permite al usuario activar una opción o no. Para crear un checkbox empleamos la etiqueta input con el atributo type="checkbox". Es importante colocar una etiqueta (label) y enlazarla con el atributo for al checkbox a través de su id, donde el valor del for y el valor del id deben coincidir.
